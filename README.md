# HQ Timetable

A beautiful course timetable web application with glassmorphism UI design.

## Features

- 🎨 Apple-style glassmorphism design
- 📅 Week view & Month view
- 📍 High current date highlight
- 📱 Mobile-first responsive design
- 🌓 Dark mode support
- 🗂️ Module class lookup

## Usage

Open `index.html` in any modern web browser.

## Setup

Just double-click `index.html` to open it.
